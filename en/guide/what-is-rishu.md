# What is Rishu made of?

Rishu is a bot that combines multiple framework functionalities, allowing it to have both bot features and human-like chat capabilities.

Rishu = Prompts + MaiBot + AstrBot + Multiple LLMs.

Where:

- **MaiBot** is a framework **born for human-like bots**.
- **AstrBot** is also an LLM-based framework, but Rishu mainly uses its plugins.

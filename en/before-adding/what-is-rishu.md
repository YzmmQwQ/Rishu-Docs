# What is Rishu?

Rishu is a bot that combines multiple frameworks, giving it both bot functionality and human-like chat capabilities.

Rishu = Prompts + MaiBot + AstrBot + Multiple LLMs.

## MaiBot

[**MaiBot**](https://docs.mai-mai.org) is a framework **born for human-like bots**.

Its design philosophy is rooted in "warmth, authenticity, and pure connection" — not pursuing perfection or efficiency, but dedicated to simulating real human interaction styles and becoming a member of social groups.

Core features:

- **Bionic thinking**: Modular architecture based on cognitive science, understanding when to speak with appropriate actions
- **Learning mechanism**: Imitates speaking styles, learns internet slang, automatically captures stickers
- **Emotion system**: Independent emotion modeling and sticker interaction capabilities
- **Memory system**: Records communication events and human expression habits, maintains intimacy with users

## AstrBot

[**AstrBot**](https://github.com/Soulter/AstrBot) is a cross-platform A.I. assistant framework, [website](https://astrbot.app).

It is defined as an intelligent agent that can run directly in various chat applications, achieving automation and intelligent collaboration through flexible agent orchestration. Rishu mainly uses its plugin system for more features.

Core features:

- **Plugin architecture**: A massive ecosystem with 1000+ plugins
- **MCP support**: Native support for Model Context Protocol, standardized external capability integration
- **Security sandbox**: Shipyard Neo provides isolated code execution environment
- **Multi-platform support**: QQ, WeChat, WeCom, Feishu, DingTalk, Telegram, Discord, Slack, etc.

## Why Combine?

Rishu combines the strengths of both frameworks:

- **MaiBot** gives Rishu a human-like soul, making conversations more natural and warm
- **AstrBot** gives Rishu powerful feature extensibility, from tool calls to knowledge base retrieval

With carefully designed prompts and multiple LLM models, Rishu is both a "friend" who can chat with you and an "assistant" who can help you handle tasks.

# Rishu 是什么？

Rishu 是混合多个框架功能的机器人，这让他同时兼顾了多个机器人功能和拟人化的聊天。

Rishu = 提示词 + MaiBot + AstrBot + 多个 LLM。

## MaiBot

[**MaiBot**](https://docs.mai-mai.org) 是一个**为拟人化机器人而生**的框架。

它的设计理念根植于"温暖、真实与纯粹的连接"——不追求完美，也不刻意追求效率，而是致力于模拟真实人类的交互风格，成为社交群体中的一员。

核心特性：

- **仿生思考**：基于认知科学理论的模块化架构，能够理解说话的时机并配合适当的动作
- **学习机制**：模仿群友的说话风格、学习网络黑话、自动捕获表情包
- **情感系统**：独立的情绪建模与表情包互动能力
- **记忆系统**：记录交流事件与人类表达习惯，维护与用户的亲密度

## AstrBot

[**AstrBot**](https://github.com/Soulter/AstrBot) 是一个跨平台的 A.I. 助手框架，[官网](https://astrbot.app)。

它被定义为一款能够直接运行在各类聊天应用中的智能代理，通过灵活的代理编排实现自动化与智能协作。Rishu 主要使用其插件系统来实现更多功能。

核心特性：

- **插件化架构**：拥有 1000+ 插件的庞大生态系统
- **MCP 支持**：原生支持 Model Context Protocol，可标准化接入外部能力
- **安全沙盒**：Shipyard Neo 提供隔离的代码执行环境
- **多平台支持**：QQ、微信、企业微信、飞书、钉钉、Telegram、Discord、Slack 等

## 为什么结合？

Rishu 结合了这两个框架的优势：

- **MaiBot** 赋予了 Rishu 拟人化的灵魂，让对话更自然、更有温度
- **AstrBot** 赋予了 Rishu 强大的功能扩展能力，从工具调用到知识库检索

再配合精心设计的提示词和多个 LLM 模型，Rishu 既是一个能陪你聊天的"朋友"，也是一个能帮你处理事务的"助手"。

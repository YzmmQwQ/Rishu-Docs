# Rishu を追加する前に

::: warning 注意
Rishu は A.I. ですので、回答に誤りや誤解を招く内容が含まれる可能性があります。Rishu の回答を参考にする前に、慎重に評価してください。
:::

## アカウントステータスについて

Rishu は通常の QQ アカウントを使用してより多くの機能を実現していますが、QQ 公式がボットフレームワークに対して厳しく管理しているため、追加時に以下が表示されることがあります：

- 「このアカウントは多数のユーザーから通報されており、セキュリティリスクがあります」
- 「このアカウントのステータスが異常です」

![セキュリティリスク警告](/img/s1.jpg)

![アカウントステータス異常](/img/s2.png)

セキュリティリスクを自己判断し、追加するかどうかを選択してください。

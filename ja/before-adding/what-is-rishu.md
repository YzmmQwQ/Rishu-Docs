# Rishu とは？

Rishu は複数のフレームワークの機能を組み合わせたボットで、ボット機能と人間らしいチャットの両方を兼ね備えています。

Rishu = プロンプト + MaiBot + AstrBot + 複数の LLM。

## MaiBot

[**MaiBot**](https://docs.mai-mai.org) は**人間らしいボットのために生まれた**フレームワークです。

その設計理念は「温かさ、真実さ、純粋な繋がり」に根ざしています——完璧を求めず、効率を追わず、本物の人間のような交流スタイルをシミュレートし、社交グループの一員になることを目指しています。

主な特徴：

- **バイオニック思考**：認知科学に基づくモジュラー構造、話すタイミングを理解し適切なアクションを組み合わせる
- **学習メカニズム**：話し方を真似、ネットスラングを学び、スタンプを自動キャプチャ
- **感情システム**：独立した感情モデリングとスタンプ連携機能
- **記憶システム**：交流イベントと人間の表現習慣を記録、ユーザーとの親密度を維持

## AstrBot

[**AstrBot**](https://github.com/Soulter/AstrBot) はクロスプラットフォームの A.I. アシスタントフレームワークです、[公式サイト](https://astrbot.app)。

様々なチャットアプリで直接動作するインテリジェントエージェントとして定義され、柔軟なエージェントオーケストレーションにより自動化とインテリジェントなコラボレーションを実現します。Rishu は主にそのプラグインシステムを使用してより多くの機能を実現しています。

主な特徴：

- **プラグインアーキテクチャ**：1000以上のプラグインを持つ巨大なエコシステム
- **MCP サポート**：Model Context Protocol のネイティブサポート、外部機能の標準化された統合
- **セキュリティサンドボックス**：Shipyard Neo が分離されたコード実行環境を提供
- **マルチプラットフォーム対応**：QQ、WeChat、WeCom、Feishu、DingTalk、Telegram、Discord、Slack など

## なぜ組み合わせるの？

Rishu は両方のフレームワークの強みを組み合わせています：

- **MaiBot** は Rishu に人間らしい魂を与え、会話をより自然で温かいものに
- **AstrBot** は Rishu に強力な機能拡張性を与え、ツール呼び出しからナレッジベース検索まで

慎重に設計されたプロンプトと複数の LLM モデルにより、Rishu はチャットできる「友達」であり、タスク処理を手伝う「アシスタント」でもあります。
